## Laravel Backend

## Проект делался при наличии laravel Herd

### 1. Клонирование и установка
```bash
git clone https://github.com/ktotopes/tonerTestVue
```
```bash
cd tonerTestVue
```
```bash
npm install
```
```bash
cp .env.example .env
```
### 2. Настройка .env

```bash
VITE_REDIRECT_URI= http://localhost:5175/callback - client url
```

npm run dev
